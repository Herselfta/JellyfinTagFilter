# 贡献指南

感谢您对本项目的关注！

## 🐛 报告问题

如果遇到问题，请提供以下信息：

1. **环境信息**
   - 操作系统和版本
   - 浏览器和版本
   - Jellyfin 服务器版本

2. **问题描述**
   - 具体的错误信息
   - 重现步骤
   - 预期行为
   - 实际行为

3. **浏览器控制台**
   - 按 F12 打开开发者工具
   - 复制 Console 中的错误信息

## 💡 建议新功能

欢迎提出功能建议，请说明：
- 功能用途和使用场景
- 为什么需要这个功能
- 如何使用这个功能

## 🔧 代码贡献

本项目主要是单文件 HTML 应用，如需修改：

1. **修改 `jellyfin_tag_filter_pwa.html`**
2. **测试功能**
   - 在多个浏览器测试
   - 在手机端测试
   - 检查控制台是否有错误

3. **更新文档**
   - 更新 README.md（如有必要）
   - 更新 CHANGELOG.md
   - 更新相关文档

4. **提交更改**
   - 提供清晰的提交信息
   - 说明修改内容和原因

## 📝 代码规范

- 使用 4 空格缩进
- 变量命名使用 camelCase
- 函数命名清晰明确
- 添加必要的注释
- 保持代码简洁可读
